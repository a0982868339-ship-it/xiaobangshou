# 小帮手项目 - 支付接入方案

## 📋 背景说明

**支付牌照问题**:
- ❌ 个人无法申请支付牌照
- ❌ 小企业很难获得支付牌照
- ✅ 必须使用第三方支付服务

**法律要求**:
- 只有持牌支付机构才能直接从事支付业务
- 普通商户必须通过持牌机构接入支付

## ✅ 方案一：官方渠道（最推荐）

### 1.1 微信支付官方接入

**官网**: https://pay.weixin.qq.com/

**所需资质**:
- ✅ 营业执照（企业/个体工商户）
- ✅ 对公银行账户（企业）或法人银行卡（个体户）
- ✅ 法人身份证
- ✅ 经营场所照片

**费用**:
- 开通费: 0元（已取消）
- 手续费: 0.6%/笔
- 提现费: 免费

**到账时间**: T+1（第二天）

**接入流程**:
```
1. 注册微信支付商户号
2. 提交资质审核（3-5个工作日）
3. 签约开通
4. 配置支付参数
5. 技术对接（API）
6. 测试上线
```

**技术实现**:
```javascript
// 后端接入示例（Node.js）
const WxPay = require('wechatpay-node-v3');

const pay = new WxPay({
  appid: '你的appid',
  mchid: '商户号',
  publicKey: '公钥',
  privateKey: '私钥'
});

// 创建订单
async function createWxPayOrder(orderInfo) {
  const result = await pay.transactions_native({
    description: orderInfo.serviceName,
    out_trade_no: orderInfo.orderNo,
    amount: {
      total: Math.round(orderInfo.amount * 100) // 转为分
    },
    notify_url: 'https://yourdomain.com/api/payment/wechat/notify'
  });
  return result.code_url; // 返回二维码链接
}
```

```javascript
// 后端接入示例
const AlipaySdk = require('alipay-sdk').default;

const alipaySdk = new AlipaySdk({
  appId: '你的appId',
  privateKey: '应用私钥',
  alipayPublicKey: '支付宝公钥'
});

// 创建支付订单
async function createAlipayOrder(orderInfo) {
  const result = await alipaySdk.exec('alipay.trade.page.pay', {
    bizContent: {
      out_trade_no: orderInfo.orderNo,
      total_amount: orderInfo.amount,
      subject: orderInfo.serviceName,
      product_code: 'FAST_INSTANT_TRADE_PAY'
    }
  });
  return result; // 返回支付表单
}
```

- ✅ 费率最低（0.6%）

## ⚠️ 方案三：第四方支付（不推荐）

### 3.1 什么是第四方支付

第四方支付是指未获得支付牌照，通过持牌机构的个人账户或商户账户进行资金中转的支付服务。

**特点**:
- 个人可用
- 门槛低
- 费率高（2-5%）
- ⚠️ 有法律风险

**风险**:
- ❌ 资金安全无保障
- ❌ 可能涉嫌"二清"（违法）
- ❌ 账户随时可能被封
- ❌ 用户投诉无门

**建议**: ❌ **不推荐使用，有法律风险！**

## 📝 技术实现建议

### 1. 设计支付表

```sql
CREATE TABLE `payments` (
  `id` BIGINT UNSIGNED NOT NULL AUTO_INCREMENT,
  `order_id` BIGINT UNSIGNED NOT NULL COMMENT '订单ID',
  `user_id` BIGINT UNSIGNED NOT NULL COMMENT '用户ID',
  `payment_no` VARCHAR(64) NOT NULL COMMENT '支付流水号',
  `payment_method` VARCHAR(20) NOT NULL COMMENT '支付方式: wechat/alipay/balance',
  `amount` DECIMAL(10,2) NOT NULL COMMENT '支付金额',
  `status` TINYINT NOT NULL DEFAULT 0 COMMENT '状态: 0-待支付, 1-支付成功, 2-支付失败',
  `paid_at` DATETIME DEFAULT NULL COMMENT '支付时间',
  `transaction_id` VARCHAR(64) DEFAULT NULL COMMENT '第三方交易号',
  `notify_data` JSON DEFAULT NULL COMMENT '支付回调数据',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_payment_no` (`payment_no`),
  KEY `idx_order_id` (`order_id`),
  KEY `idx_user_id` (`user_id`),
  KEY `idx_status` (`status`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='支付记录表';
```

### 2. 支付接口设计

```javascript
// backend/src/controllers/paymentController.js

// 创建支付
exports.createPayment = async (req, res) => {
  try {
    const { orderId, paymentMethod } = req.body;
    const userId = req.user.id;
    
    // 1. 获取订单信息
    const [orders] = await db.query('SELECT * FROM orders WHERE id = ? AND user_id = ?', [orderId, userId]);
    if (orders.length === 0) return sendError(res, '订单不存在');
    const order = orders[0];
    
    if (order.pay_status === 1) return sendError(res, '订单已支付');
    
    // 2. 生成支付流水号
    const paymentNo = 'PAY' + Date.now() + Math.floor(Math.random() * 1000);
    
    // 3. 根据支付方式处理
    let paymentData;
    if (paymentMethod === 'wechat') {
      // 调用微信支付
      paymentData = await createWxPayOrder(order);
    } else if (paymentMethod === 'alipay') {
      // 调用支付宝
      paymentData = await createAlipayOrder(order);
    } else if (paymentMethod === 'balance') {
      // 余额支付
      paymentData = await payWithBalance(userId, order);
    }
    
    // 4. 记录支付记录
    await db.query(
      'INSERT INTO payments (order_id, user_id, payment_no, payment_method, amount) VALUES (?, ?, ?, ?, ?)',
      [orderId, userId, paymentNo, paymentMethod, order.total_price]
    );
    
    sendSuccess(res, { paymentNo, ...paymentData });
  } catch (error) {
    console.error('创建支付失败:', error);
    sendError(res, error.message);
  }
};

// 支付回调
exports.paymentNotify = async (req, res) => {
  try {
    // 1. 验证签名（防止伪造）
    const isValid = verifySignature(req.body);
    if (!isValid) return res.status(400).send('FAIL');
    
    // 2. 更新支付状态
    const { out_trade_no, transaction_id } = req.body;
    
    await db.query(
      'UPDATE payments SET status = 1, paid_at = NOW(), transaction_id = ? WHERE payment_no = ?',
      [transaction_id, out_trade_no]
    );
    
    // 3. 更新订单状态
    await db.query(
      'UPDATE orders SET pay_status = 1, pay_time = NOW() WHERE order_no = (SELECT o.order_no FROM orders o JOIN payments p ON o.id = p.order_id WHERE p.payment_no = ?)',
      [out_trade_no]
    );
    
    // 4. 返回成功
    res.send('SUCCESS');
  } catch (error) {
    console.error('支付回调处理失败:', error);
    res.status(500).send('FAIL');
  }
};
```
