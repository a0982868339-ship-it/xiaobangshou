### 3. 订单管理模块

#### 3.1 创建订单
```
POST /api/orders
```
**状态**: ✅ 完全可用
- 支持标准服务订单
- 自动生成订单号
- 计算订单金额
- 事务处理保证数据一致性

**必需字段**:
- serviceId: 服务ID
- serviceTime: 预约时间
- serviceAddress: 服务地址
- contactName: 联系人
- contactPhone: 联系电话

**可选字段**:
- remark: 备注说明
- addressId: 地址ID

**测试结果**:
```json
{
  "orderId": 3,
  "orderNo": "ORD1768636760952389"
}
```

#### 3.2 获取订单列表
```
GET /api/orders
GET /api/orders?status=0
```
**状态**: ✅ 完全可用
- 获取当前用户的订单
- 支持按状态筛选
- 按创建时间倒序

**订单状态**:
- 0: 待接单
- 1: 已接单
- 2: 服务中
- 3: 已完成
- 4: 已取消

#### 3.3 订单详情
```
GET /api/orders/:id
```
**状态**: ✅ 可用
- 返回完整订单信息
- 包含服务信息
- 包含用户信息

#### 3.4 取消订单
```
POST /api/orders/:id/cancel
```
**状态**: ✅ 可用
- 只能取消待接单状态的订单
- 记录取消原因
