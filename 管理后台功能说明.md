## 📊 数据总览（Dashboard）

### 1. 顶部数据卡片（4个渐变卡）

**卡片1 - 服务项目**（蓝紫渐变）
- 显示：26个服务项目
- 趋势：较上月 +12%
- 点击：跳转到服务管理

**卡片2 - 注册用户**（粉红渐变）
- 显示：总用户数
- 趋势：较上月 +25%
- 点击：跳转到用户管理

**卡片3 - 今日订单**（蓝色渐变）
- 显示：今日订单数
- 趋势：较昨日 +8%

**卡片4 - 今日收益**（绿色渐变）
- 显示：今日收益金额
- 趋势：较昨日 +15%

### 2. 订单趋势图（折线图）

- **位置**：中间大图
- **功能**：
  - 显示近7天/近30天订单趋势
  - 渐变区域填充
  - 鼠标悬停显示详细数据
- **交互**：
  - 切换"近7天"/"近30天"查看不同周期

### 3. 服务分布图（饼图）

- **位置**：右侧
- **功能**：
  - 展示6大分类的服务占比
  - 彩色环形图
  - 悬停放大效果
- **数据**：
  - 陪伴照护：8个服务
  - 宠物服务：6个服务
  - 跑腿代办：4个服务
  - 居家服务：5个服务
  - 健康养生：2个服务
  - 教育培训：1个服务

### 4. 热门服务 Top 10

- 排名展示
- 服务名称、价格、销量
- 点击"查看全部"跳转服务管理

### 5. 实时数据面板

**大数据展示**：
- 总用户数 + 新增用户
- 服务者数 + 待审核

**快速统计**（4个指标）：
- 今日订单
- 今日收益
- 活跃服务者
- 待处理投诉

**系统信息**：
- 系统运行状态 ✅
- 数据库统计 ℹ️
- 待开发功能 ⚠️

## 👥 用户管理（Users）

### 功能特性

**顶部统计卡**（4个）：
1. 总用户数
2. 已实名用户数
3. 今日新增
4. 会员用户数

**搜索和筛选**：
- 🔍 搜索：手机号/昵称
- 筛选：用户类型（普通/服务者/兼是）
- 筛选：实名状态（已实名/未实名）
- 筛选：账户状态（正常/禁用）
- 📥 导出用户数据

**用户列表展示**：
| 列 | 内容 |
|---|---|
| ID | 用户ID |
| 用户信息 | 头像 + 昵称 + 手机号 |
| 用户类型 | 普通用户/服务者/兼是（标签）|
| 实名认证 | 已认证/未认证（标签）|
| 余额 | 红色显示金额 |
| 积分 | 积分数值 |
| 状态 | 正常/禁用（标签）|
| 注册时间 | 格式化时间 |
| 操作 | 详情/编辑/禁用按钮 |

**详情对话框**：
- 完整用户信息
- 真实姓名、性别
- 账户余额、积分
- 注册时间、最后登录
- 实名认证状态

**操作功能**：
- 📝 查看详情
- ✏️ 编辑用户信息
- 🚫 禁用/启用账户
- 📥 导出用户数据

## 📝 服务管理（Services）

**功能**：
- 查看所有26个服务
- 搜索服务
- 显示价格、销量、状态
- 分页查询

## 📁 分类管理（Categories）

**功能**：
- 查看所有分类
- 分类状态管理
- 排序显示

## 📦 套餐管理（Packages）

**功能**：
- 查看12个套餐方案
- 显示套餐类型（包月/次卡）
- 价格对比（原价vs套餐价）

**功能**：
- 待审核服务者列表
- 查看认证详情（身份证、人脸、健康证等）
- 通过/拒绝审核
- 审核日志记录
