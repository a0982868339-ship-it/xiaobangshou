# 服务区域设置方案

## 🎯 业务场景分析

### 本地生活服务的地理特点

1. **服务半径有限**：一般在10-30公里内
2. **区域差异大**：同城不同区距离可能很远
3. **交通成本**：路程远影响接单意愿
4. **时间成本**：往返时间影响效率

---

## 📊 方案对比

### 方案1：城市+区县选择（推荐）⭐⭐⭐⭐⭐

**适用场景**：北京、上海等大城市

**实现方式**：
```
常驻城市: 北京市
接单范围:
  ☑️ 覆盖全城
  或
  ☑️ 朝阳区
  ☑️ 海淀区
  ☑️ 东城区
```

**优点**：
- ✅ 清晰明确
- ✅ 操作简单
- ✅ 匹配准确
- ✅ 无需GPS

**缺点**：
- ⚠️ 大区域内可能距离很远
- ⚠️ 边界问题（如朝阳和海淀交界）

---

### 方案2：距离半径（推荐）⭐⭐⭐⭐⭐

**适用场景**：所有城市

**实现方式**：
```
常驻地址: 北京市朝阳区望京SOHO
接单范围:
  ○ 5公里内
  ● 10公里内
  ○ 20公里内
  ○ 30公里内
  ○ 不限距离
```

**优点**：
- ✅ 精确控制服务范围
- ✅ 减少交通成本
- ✅ 提高接单效率
- ✅ 用户体验好（知道服务者距离）

**缺点**：
- ⚠️ 需要GPS定位
- ⚠️ 需要计算距离

---

### 方案3：混合模式（最佳）⭐⭐⭐⭐⭐

**结合方案1和方案2**

**实现方式**：
```
第一层：城市选择
  常驻城市: 北京市

第二层：区域选择
  ☑️ 覆盖全城
  或
  ☑️ 朝阳区、海淀区、东城区

第三层：距离限制（可选）
  接单距离: 15公里内
```

**优点**：
- ✅ 灵活性最强
- ✅ 适应不同场景
- ✅ 用户可自主选择
- ✅ 匹配精度高

### 2. 区县多选实现

```vue
<template>
  <div class="area-page">
    <van-nav-bar title="服务区域" left-arrow @click-left="$router.back()" />
    
    <!-- 常驻城市 -->
    <van-cell-group inset title="当前服务城市">
      <van-cell title="常驻城市" is-link :value="selectedCity" @click="showCityPicker = true" />
    </van-cell-group>

    <!-- 接单范围 -->
    <van-cell-group inset title="接单范围">
      <van-cell title="覆盖全城">
        <template #right-icon>
          <van-switch v-model="allCity" size="24" @change="onAllCityChange" />
        </template>
      </van-cell>
      
      <!-- 区县选择 -->
      <van-cell v-if="!allCity" title="选择区县" is-link @click="showDistrictPicker = true">
        <template #value>
          <span style="color: #969799;">已选{{ selectedDistricts.length }}个</span>
        </template>
      </van-cell>
      
      <!-- 已选区县展示 -->
      <van-cell v-if="!allCity && selectedDistricts.length > 0">
        <div class="district-tags">
          <van-tag 
            v-for="item in selectedDistricts" 
            :key="item.code"
            closeable
            type="primary"
            @close="removeDistrict(item.code)"
            style="margin: 4px;"
          >
            {{ item.name }}
          </van-tag>
        </div>
      </van-cell>
    </van-cell-group>

    <!-- 距离限制（可选） -->
    <van-cell-group inset title="距离限制（可选）">
      <van-cell title="最远接单距离">
        <template #value>
          <van-stepper v-model="maxDistance" step="5" min="5" max="50" />
          <span style="margin-left: 8px; color: #969799;">公里</span>
        </template>
      </van-cell>
      <van-cell>
        <template #title>
          <span style="font-size: 12px; color: #969799;">
            💡 设置后，系统只推送该距离内的订单
          </span>
        </template>
      </van-cell>
    </van-cell-group>

    <!-- 保存按钮 -->

<script setup>
import { ref, computed } from 'vue'
import { showSuccessToast } from 'vant'
import { areaList } from '@vant/area-data'

const selectedCity = ref('北京市')
const allCity = ref(true)
const selectedDistricts = ref([])
const maxDistance = ref(15) // 默认15公里
const showCityPicker = ref(false)
const showDistrictPicker = ref(false)

// 区县数据（根据城市动态生成）
const districtColumns = computed(() => {
  // 北京市的区县
  const beijingDistricts = [
    { text: '东城区', value: '110101' },
    { text: '西城区', value: '110102' },
    { text: '朝阳区', value: '110105' },
    { text: '丰台区', value: '110106' },
    { text: '石景山区', value: '110107' },
    { text: '海淀区', value: '110108' },
    { text: '门头沟区', value: '110109' },
    { text: '房山区', value: '110111' },
    { text: '通州区', value: '110112' },
    { text: '顺义区', value: '110113' },
    { text: '昌平区', value: '110114' },
    { text: '大兴区', value: '110115' },
    { text: '怀柔区', value: '110116' },
    { text: '平谷区', value: '110117' },
    { text: '密云区', value: '110118' },
    { text: '延庆区', value: '110119' }
  ]
  
  // TODO: 根据selectedCity返回对应城市的区县
  return beijingDistricts
})

const onAllCityChange = (val) => {
  if (val) {
    selectedDistricts.value = []
  }
}

const onConfirmCity = (values) => {
  selectedCity.value = values[1].name
  showCityPicker.value = false
  // 切换城市时清空区县选择
  selectedDistricts.value = []
}

const onConfirmDistrict = ({ selectedOptions }) => {
  const district = selectedOptions[0]
  // 检查是否已选择
  if (!selectedDistricts.value.find(d => d.code === district.value)) {
    selectedDistricts.value.push({
      code: district.value,
      name: district.text
    })
  }
  showDistrictPicker.value = false
}

const removeDistrict = (code) => {
  selectedDistricts.value = selectedDistricts.value.filter(d => d.code !== code)
}

const saveArea = async () => {
  try {
    const request = (await import('../api')).default
    await request.put('/providers/service-areas', {
      serviceAreas: {
        city: selectedCity.value,
        allCity: allCity.value,
        districts: selectedDistricts.value,
        maxDistance: maxDistance.value
      }
    })
    
    localStorage.setItem('provider_service_area', JSON.stringify({
      city: selectedCity.value,
      allCity: allCity.value,
      districts: selectedDistricts.value,
      maxDistance: maxDistance.value
    }))
    
    showSuccessToast('保存成功')
  } catch (error) {
    console.error('保存失败:', error)
  }
}
</script>

// 使用浏览器地理位置API（免费）
navigator.geolocation.getCurrentPosition(
  (position) => {
    const { latitude, longitude } = position.coords
    console.log('位置:', latitude, longitude)
  },
  (error) => {
    console.error('定位失败:', error)
  }
)

// 计算两点距离（Haversine公式）
function calculateDistance(lat1, lon1, lat2, lon2) {
  const R = 6371; // 地球半径（公里）
  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLon = (lon2 - lon1) * Math.PI / 180;
  const a = Math.sin(dLat/2) * Math.sin(dLat/2) +
            Math.cos(lat1 * Math.PI / 180) * Math.cos(lat2 * Math.PI / 180) *
            Math.sin(dLon/2) * Math.sin(dLon/2);
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
  return R * c; // 返回公里数
}
```

**优点**：
- 免费（浏览器原生支持）
- 精确匹配
- 智能派单
