2. **登录账号**:
   - 点击右下角"我的"

## 🐛 常见问题调试

### 问题 1: 网络错误

**症状**: 前端显示"网络错误"
**原因**: 后端服务停止或崩溃
**解决**:
```bash
# 查看后端日志
tail -50 logs/backend.log

# 重启后端
cd backend && npm run dev
```

### 问题 2: 接口返回 401

**症状**: API 返回"未登录或token已过期"
**原因**: Token 失效或未登录
**解决**: 重新登录获取新 Token

### 问题 3: 数据库连接失败

**症状**: 后端启动报错"❌ 数据库连接失败"
**原因**: MySQL 未启动或密码错误
**解决**:
```bash
# 启动 MySQL
brew services start mysql

# 检查密码
vim backend/.env
# 确保 DB_PASSWORD=root
```

### 问题 4: 端口被占用

**症状**: 启动失败，提示端口被占用
**解决**:
```bash
# 查看占用端口的进程
lsof -i :3000
lsof -i :9000

# 杀死进程
kill -9 <PID>
```

- **Console**: 查看前端报错
- **Network**: 查看 API 请求状态
- **Application**: 查看 LocalStorage 中的 Token

### 2. 后端调试

在 `backend/src/` 任意文件中添加:
```javascript
console.log('调试信息:', data)
```
