## 🎨 设计规范

### 配色方案

**用户端主题色**：
```css
主色：#667eea → #764ba2（蓝紫渐变）
辅色：#f093fb → #f5576c（粉红渐变）
强调色：#ee0a24（红色，价格）
```

**服务者端主题色**：
```css
主色：#52c41a → #95de64（绿色渐变）
辅色：#faad14（橙色，警告）
强调色：#ee0a24（红色，收益）
```

**管理后台配色**：
```css
卡片1：#667eea → #764ba2
卡片2：#f093fb → #f5576c
卡片3：#4facfe → #00f2fe
卡片4：#43e97b → #38f9d7
```

### 圆角规范
```css
小圆角：8px（标签、按钮）
中圆角：12px（卡片、输入框）
大圆角：16-20px（头部区域）
完整圆角：50%（头像、图标）
```

### 阴影规范
```css
轻阴影：0 2px 8px rgba(0, 0, 0, 0.06)
中阴影：0 4px 12px rgba(0, 0, 0, 0.1)
重阴影：0 8px 24px rgba(0, 0, 0, 0.15)
彩色阴影：0 4px 20px rgba(82, 196, 26, 0.2)
```

### 动画效果
```css
淡入淡出：fadeIn/fadeOut 0.3s
滑动：slideIn 0.4s
缩放：scale(0.95-1) 0.3s
悬停上浮：translateY(-5px) 0.3s
```

## 📊 技术实现

### CSS技巧
```css
/* 渐变背景 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 毛玻璃 */
backdrop-filter: blur(10px);
background: rgba(255, 255, 255, 0.2);

/* 渐变文字 */
background: linear-gradient(135deg, #52c41a, #95de64);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

/* 卡片阴影 */
box-shadow: 0 4px 20px rgba(0, 0, 0, 0.08);

/* 动画 */
animation: fadeInUp 0.8s;
transition: all 0.3s;
```

### 动画库
```css
@keyframes fadeInDown {
  from { opacity: 0; transform: translateY(-30px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes slideInUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}
```
