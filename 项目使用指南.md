## 📱 一、移动端H5 - 用户端试用

### 访问方式

**方法1：电脑浏览器访问（推荐）**
```
1. 打开Chrome浏览器
2. 访问: http://localhost:9000
3. 按F12打开开发者工具
4. 点击"切换设备模拟"图标（或Ctrl+Shift+M）
5. 选择设备: iPhone 12 Pro 或 iPhone SE
```

**方法2：手机访问**
```
1. 确保手机和电脑在同一WiFi
2. 查看电脑IP: 
   Mac: ifconfig | grep "inet " | grep -v 127.0.0.1
   Windows: ipconfig | findstr IPv4
3. 手机浏览器访问: http://你的IP:9000
   例如: http://192.168.1.100:9000
```

## 🐛 常见问题

### 1. 页面空白
```
解决方案:
- 检查服务是否启动
- 查看浏览器控制台错误
- 刷新页面（Ctrl+F5）
```

### 2. 验证码收不到
```
说明:
- 当前为开发模式
- 验证码直接显示在页面上
- 不会真实发送短信
```

### 3. 订单创建失败
```
检查:
- 是否已登录
- 是否已添加地址
- 网络请求是否成功
```

### 4. 图片不显示
```
说明:
- 当前使用占位图
- 可以在管理后台上传真实图片
```
