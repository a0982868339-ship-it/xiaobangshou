## 📱 如何封装成App

### 最快速（立即测试）
**添加到主屏幕**：
1. 手机浏览器访问 http://你的IP:9000（用户端）
2. 手机浏览器访问 http://你的IP:9001（服务者端）
3. 点击"添加到主屏幕"
4. 桌面图标像真App一样使用

### 正式打包（Cordova）
```bash
# 安装Cordova
npm install -g cordova

# 创建用户端App
cordova create user-app com.xiaobangshou.user 小帮手
cd user-app
cordova platform add android ios

# 构建H5并复制
cd ../mobile-user
npm run build
cp -r dist/* ../user-app/www/

# 打包
cd ../user-app
cordova build android
```

详见：**[HOW_TO_MAKE_APP.md](HOW_TO_MAKE_APP.md)**
